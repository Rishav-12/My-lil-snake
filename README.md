# snake-game
A Fun Project/Game

### Motivation
Working with pygame to make simple 2D games

### Requirements
* Python
* Pygame

### Description
* Built using Pygame

### Usage
* Run 
```console
 pip install -r requirements.txt 
```
* Run 
```console
python main.py
```
or
```console
python3 main.py
```
* The game itself follows the basic snake game rules
    * Each food gives you 10 points
    * You also have a local High Score
